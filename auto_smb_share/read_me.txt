# Samba Script

This Bash script automates the setup of a Samba file server on Linux systems. 
Samba is used to share files and printers between Linux and Windows systems over a network.


## How to Use

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/trix899/scripts_linux/
   cd scripts_linux/auto_smb_share

3. chmod +x auto_smb_share.sh (file is then executable) 

4.sudo ./auto_smb_share.sh    #if another user
  ./auto_smb_share.sh         #if you are root user

5 Follow the steps to make a new share/users.
